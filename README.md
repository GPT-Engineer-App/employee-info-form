# employee-info-form

Create a single page application with a form in first page which takes employee information and on the submit button click, navigates to another page where all the entered information is shown in read only mode.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/employee-info-form.git
cd employee-info-form
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
